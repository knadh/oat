Third-party extensions that add optional functionality to Oat without bloating the core bundle.

- **[oat-chips](https://github.com/someshkar/oat-chips)**: Chip/tag component with dismissible filters, colors, and toggle selection. ~1KB gzipped. [Demo](https://oat-chips.somesh.dev)

Have an extension? [Open a PR](https://github.com/knadh/oat) to add it here.
