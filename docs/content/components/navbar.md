Use semantic `<nav>` element with `.navbar` class.

{% demo() %}
```html
<nav class="navbar">
  <a href="/" class="brand">Logo</a>
  <button class="toggle"></button>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>
```
{% end %}
